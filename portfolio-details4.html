Project Info

  • Domain: Data Science / Music Analytics
  • Stack: Python, scikit-learn, Pandas, NumPy, Matplotlib
  • Models: Linear Regression, Random Forest, Ridge, Lasso
  • GitHub: View Repository

The Problem

Music labels and independent artists have no reliable way to forecast a song's streaming performance before release — making it hard to allocate promotion budgets effectively.

What I Built

  • Trained and benchmarked four regression models (Linear, Random Forest, Ridge, Lasso) on Spotify song attributes including genre, artist popularity, tempo, and duration
  • Tuned hyperparameters to achieve a 20% accuracy improvement across models
  • Applied feature importance analysis, reducing model complexity by 15% while maintaining prediction quality
  • Identified Random Forest as the top performer at 85% prediction accuracy

Impact

Produced a reproducible, extensible prediction pipeline with clear model comparison metrics — directly applicable to music recommendation and playlist curation systems.
