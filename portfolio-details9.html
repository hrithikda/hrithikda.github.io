AI Financial Analyst Agent

Overview

A LangGraph-based multi-agent system that autonomously researches public companies by ingesting SEC filings (10-K, 10-Q), earnings call transcripts, and real-time price data — then synthesizes them into a structured investment research report with thesis, risks, and key financial metrics.

What was built

  • Multi-agent pipeline: LangGraph orchestration with specialist agents for document retrieval, financial ratio extraction, sentiment analysis of earnings transcripts, and report synthesis.
  • RAG retrieval: ChromaDB vector store with hybrid BM25 + dense retrieval over chunked SEC filings and transcripts. Metadata filtering by fiscal year and document type.
  • Financial extraction: Structured output extraction (revenue, margins, FCF, D/E ratio, YoY growth) from unstructured filing text using Claude API with schema enforcement.
  • Report generation: Templated markdown reports covering investment thesis, bull/bear case, risk factors, and comparable company context — generated per-ticker on demand.
  • FastAPI backend: REST endpoints for ticker lookup, report generation, and cached report retrieval. Streamlit frontend for interactive querying.

Why it matters

Financial research requires synthesizing information across dozens of documents with strict factual accuracy — a hard test for RAG systems. This project shows how structured output enforcement, source citation, and agent specialization combine to produce research that's traceable back to source documents, not just plausible-sounding LLM output.
